시스템 전체 흐름

지사 계정과 본사 계정은 동일한 로그인 페이지를 사용하며, 로그인 후 권한에 따라 화면이 다르게 표시됩니다.
지사 사용 가이드 개요

로그인 및 시작

  • 로그인 페이지에서 지사 계정으로 로그인합니다.
  • 로그인 후 1단계 거래명세서 업로드 화면이 표시됩니다.
  • 배정 목록에서 본사가 배정한 고객을 확인할 수 있습니다.

문서 상태 흐름

작성 필요 → 예약 접수 중 → 예약 확정 → 시공 완료
  • 작성 필요: 본사 배정 완료, 아직 작성 시작 전
  • 예약 접수 중: 4단계까지 저장 완료 (draft 상태)
  • 예약 확정: 5단계에서 [예약 확정] 버튼 클릭 후
  • 시공 완료: 5단계에서 [시공 완료] 버튼 클릭 후 → 6단계로 이동
1단계: 거래명세서 업로드
지사 전용

배정 목록 확인

  • 화면 상단에 본사가 배정한 고객 목록이 표시됩니다.
  • 작성 필요 상태의 고객을 선택해 작성을 시작합니다.
  • 이미 저장된 문서는 5단계 저장 문서 관리에서 확인합니다.

고객 정보 직접 입력

  • 배정 목록에서 고객을 선택하면 고객 정보가 자동으로 채워집니다.
  • 고객명, 연락처, 주소 등 필요한 정보를 직접 입력하거나 수정합니다.

다음 단계로

  • 고객 정보 확인 후 다음 버튼을 클릭합니다.
2단계: 제품 선택
지사 전용

브랜드별 제품 선택

  • 상단 탭에서 밀워키(Milwaukee) 또는 기아(KIA) 브랜드를 선택합니다.
  • 제품 카드를 클릭해 원하는 패키지를 선택합니다. (다중 선택 가능)
  • 3단 선반 제품은 설치 위치(좌측/우측/양측)를 추가 선택합니다.
선택한 제품은 카드 테두리가 파란색으로 표시됩니다.
3단계: 설치 정보 입력
지사 전용

입력 항목

  • 설치 날짜: 달력에서 설치 예정 날짜를 선택합니다.
  • 설치 시간: 오전/오후 + 시/분을 선택하거나 직접 입력합니다.
  • 설치 주소: 배정 정보에서 자동 입력되며, 수정 가능합니다. [고객 주소 복사] 버튼으로 자동 입력도 됩니다.
  • 특이사항: 시공 관련 메모를 입력합니다.

임시저장

  • 날짜 없이 저장 → 조율 중 상태로 저장
  • 날짜 입력 후 저장 → 예약 접수 중 상태로 저장
임시저장 후 나중에 5단계에서 [수정하기]로 다시 편집할 수 있습니다.
4단계: 최종 확인 및 저장
지사 전용

최종 확인

  • 고객 정보, 제품, 설치 정보를 최종 확인합니다.
  • 자재 점검표 체크박스를 확인합니다.
  • 시공자 이름을 입력합니다.

저장 및 발송 버튼

저장하기 이메일 발송 PDF 인쇄
  • 저장하기: Cloudflare D1 DB에 저장 + 거래명세서 이미지 R2에 저장
  • 이메일 발송: 고객에게 거래명세서 이미지 포함 이메일 발송
  • PDF 인쇄: 브라우저 인쇄 기능으로 PDF 저장 가능
저장 완료 후 자동으로 초기화되어 새 고객 접수를 바로 시작할 수 있습니다.
5단계: 저장 문서 관리
지사 전용

검색 및 필터

  • 기간 검색: 시작 날짜 ~ 종료 날짜로 필터링
  • 고객명 검색: 고객 이름, 주소, 시공자로 검색
  • 상태 필터: 전체 / 예약 접수 중 / 예약 확정 / 시공 완료

각 문서 카드 버튼

상세보기 수정하기 예약 확정 시공 완료 삭제
  • 예약 확정: draft → confirmed 상태 변경 (본사에서 확인 가능)
  • 시공 완료: confirmed → completed 상태 변경 → 5단계에서 사라지고 6단계 매출 관리로 이동
시공 완료 처리 후 해당 문서는 5단계에서 사라집니다. 6단계에서 확인하세요.

Excel 내보내기

  • 전체 문서를 Excel 파일로 다운로드합니다.
  • 파일명: PV5_시공확인서_YYYY-MM-DD.xlsx
6단계: 매출 관리
지사 전용

매출 통계 대시보드

  • 총 매출액: 시공 완료된 전체 매출 합계
  • 총 소비자 가격: 소비자가 기준 합계
  • 시공 건수: 완료 건수 및 평균 매출

기간 검색 필터

  • 이번 주 / 이번 달 / 이번 분기 / 사용자 지정 기간 선택
  • 고객명으로 추가 검색 가능

정산 처리

선택항목 일괄 정산완료 정산완료 (개별) 5단계로 수정
  • 개별 정산완료: 각 행 우측 [✅ 정산완료] 버튼 클릭 → 정산 라벨 입력 후 확인
  • 일괄 정산완료: 체크박스로 여러 건 선택 후 [선택항목 일괄 정산완료] 버튼 클릭
  • 5단계로 수정: 시공 완료를 취소하고 5단계로 되돌림
정산 라벨 입력 예시: "2026년 3월 정산" → 7단계에서 월별로 분류되어 표시됩니다.

Excel 다운로드

  • 매출 목록을 Excel로 다운로드합니다.
  • 파일명: PV5_매출관리_YYYY-MM-DD.xlsx
7단계: 정산 내역
지사 전용

월별 정산 내역 조회

  • 6단계에서 정산 완료된 항목이 월별로 그룹화되어 표시됩니다.
  • 각 그룹을 클릭하면 세부 내역(고객명, 제품, 매출, 시공자)을 확인할 수 있습니다.

월별 Excel 다운로드

  • 각 월별 그룹 옆 Excel 버튼 클릭
  • 파일명: 2026년 2월 정산.xlsx (정산 라벨 기준)

6단계로 되돌리기

  • 정산 취소가 필요한 경우 [↩ 6단계로 되돌리기] 버튼 클릭
  • 해당 항목이 다시 6단계 매출 관리로 이동됩니다.
본사 관리 페이지
본사 전용
본사 계정으로 로그인하면 전체 지사 관리 화면이 표시됩니다.
URL: https://dev-multi-tenant.pv5-webapp.pages.dev/static/hq

고객 배정 등록

  • [+ 배정 추가] 버튼 클릭 → 고객 정보 입력 → 지사 선택 → 등록
  • 등록 즉시 해당 지사 1단계 배정 목록에 표시됩니다.
  • 동일 고객 + 동일 지사 + 동일 날짜 중복 등록은 자동 차단됩니다.

전체 접수 현황 테이블

  • 모든 지사의 접수 현황을 한눈에 확인합니다.
  • 컬럼 순서: 접수일자 → 상태 → 주문자명 → 연락처 → 제품 → 지사
대기 중 조율 중 진행 중 예약 확정 시공 완료

통계 대시보드

  • 전체 건수 / 진행 중 건수 / 완료 건수를 카드 형태로 표시
  • 지사별 현황을 빠르게 파악할 수 있습니다.

상태 관리

  • 각 배정 건의 상태를 본사에서 직접 변경할 수 있습니다.
  • 지사 대리 접속: 특정 지사를 선택해 해당 지사 화면을 확인할 수 있습니다.
자주 묻는 질문 (FAQ)
Q 저장했는데 5단계 문서 목록에 안 보입니다.
A 시공 완료 처리된 문서는 5단계에서 자동으로 제외되고 6단계 매출 관리로 이동됩니다. 6단계를 확인해보세요. 그래도 없다면 로그아웃 후 재로그인하면 서버에서 새로 불러옵니다.
Q 1단계 배정 목록에서 이미 저장한 고객이 또 보입니다.
A 배정 목록은 아직 작성을 시작하지 않은 건만 표시됩니다. 이미 저장된 고객이 배정 목록에 보인다면, 5단계에서 해당 문서를 확인하고 저장이 완료됐는지 확인하세요.
Q 같은 고객이 두 번 등록됩니다.
A 동일 고객 + 동일 지사 + 동일 날짜로 등록 시 자동으로 중복이 차단됩니다. 날짜가 다르거나 다른 지사에서 등록한 경우는 별도 건으로 처리됩니다.
Q 정산 완료한 건을 다시 수정하고 싶습니다.
A 7단계 정산 내역에서 해당 건의 [↩ 6단계로 되돌리기] 버튼을 클릭하면 6단계 매출 관리로 이동됩니다. 6단계에서 [↩ 5단계로 수정] 버튼으로 추가 수정도 가능합니다.
